# MedSim Diagnostics GUI

A KivyMD landing page for the MedSim medical diagnosis simulation game.

## Setup

1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the app

```bash
python main.py
```

The landing page showcases the game's branding, key features, and a highlighted case to set the tone for future gameplay screens.
